Sistema de Gestión de Inventarios
Ingreso de Productos




Para ver, editar o borrar un registro.
  1. Ingresa el ID del producto a consultar en la barra de abajo.
  2. Presiona el botón Consultar Producto ID
  3. Si el ID del producto coincide con alguno del sistema, se desplegarán los otros botones.

Para añadir un registro.
  1. Completar los campos a continuación
  2. Presiona el botón Ingresar
  3. Comprueba tu lista de items presionando Listar todos los productos.
